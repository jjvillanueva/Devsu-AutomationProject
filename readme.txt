# Devsu-AutomationProject
Automation project for Devsu of a purchase flow on a Demo Web Page

## Descripción

Prueba funcional automatizada de un flujo de compra en la página https://www.demoblaze.com/
Dentro de las funcionalidades principales se encuentran:

1. Proceso de autenticación con un usuario legítimo
2. Proceso de flujo de compra completo

## Requisitos Previos

El proyecto se ejecutó con las siguientes herramientas instaladas (pudiera funcionar con otras versiones)

- [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/javase-downloads.html) versión 19.0.1
- [Maven](https://maven.apache.org/download.cgi)

## Configuración del entorno
1. Clona el repositorio en tu computadora local:
   git clone <repositotyLink>

## Instalación y Dependencias
Las dependencias de este proyecto son administradas por Maven.
1. Compila el proyecto y descarga las depencias con Maven:
   mvn clean install

## Run/Debug configuration
Este proyecto fue creado con JUnit como marco de pruebas, puedes configurar un Run que corra todas las pruebas
del proyecto.
1. Entra en la configuración del Run/Debug.
2. Completa los datos como Nombre del Run, version de java a utilizar y directorio de trabajo.
3. Aplica los cambios

## Ejecución del proyecto

### Desde un IDE
Este proyecto fue realizado con IntelliJ, pero debe funcionar de manera similar para otros IDE.
1. Importa el proyecto en el IDE.
2. Ejecuta el Run/Debug configuration creado anteriormente
